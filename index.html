Research Interests

I am interested in design, implementation of system-level programming languages, compilers, and formal verification for building reliable, certified software and hardware systems. A common theme is constructing correct embedded systems (i.e., control software in safety-critical systems, embedded system hardware designs) by applying formal verification including static analysis, model checking and correctness proof on the source code implementation, or mathematical models of these systems producing by model-based design toolchains, and formally certifying the correctness of the compilers and code generators in the model-based design toolchains.
